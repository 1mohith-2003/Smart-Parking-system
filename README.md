# Smart Parking System


Minimal smart parking project (Express + SQLite backend, React frontend).


## Run Backend
```bash
cd backend
npm install
npm run start
```


## Run Frontend
```bash
cd frontend
npm install
npm run dev
```


API runs default on http://localhost:3000
Frontend Vite runs on http://localhost:5173


## Features
- Manage parking slots
- Reserve / release a slot
- Simple dashboard showing occupancy


## Notes
This is a starter template. Extend with real sensors, authentication, payment, and monitoring as needed.
